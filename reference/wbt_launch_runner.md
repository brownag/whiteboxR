# Launch 'WhiteboxTools Runner' GUI

`wbt_launch_runner()`: Launch the 'WhiteboxTools Runner' GUI at
[`wbt_runner_path()`](wbt_init.md)

## Usage

``` r
wbt_launch_runner(clear_app_state = FALSE)
```

## Arguments

- clear_app_state:

  Clear application state memory? Default: `FALSE`

## Details

Opens the 'WhiteboxTools Runner' GUI included with WhiteboxTools Open
Core v2.3.0 or higher.
