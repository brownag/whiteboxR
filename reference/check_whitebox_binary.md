# Check for 'WhiteboxTools' executable path

Check for 'WhiteboxTools' executable path

## Usage

``` r
check_whitebox_binary(silent = TRUE)
```

## Arguments

- silent:

  logical. Print help on installation/setting path. Default `TRUE`.

## Value

logical if 'WhiteboxTools' executable file exists.
