# Version information for 'WhiteboxTools'

Version information for 'WhiteboxTools'

## Usage

``` r
wbt_version(extract = FALSE)
```

## Arguments

- extract:

  Extract semantic version number from first line of result? Default:
  `FALSE`

## Value

Returns the version information for 'WhiteboxTools' as an R character
vector.

## Examples

``` r
if (FALSE) { # \dontrun{
wbt_version()
} # }
```
